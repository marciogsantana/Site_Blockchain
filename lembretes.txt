## inicializar projeto:
# npm init -y   // inicia projeto
# npm install --save-dev hardhat@2.9.9   // instala dependencia do projeto
# npx hardhat // inicializa o hardhat
## selecionar projeto basico
# npm install --save-dev @nomiclabs/hardhat-waffle ethereum-waffle chai @nomiclabs/hardhat-ethers ethers   // dependencias
# npx hardhat accounts   // gerar as contas de teste
# npx hardhat compile  // compila
# npx hardhat test  // teste

## rodar uma rede local com algumas contas

# npx hardhat node

## configurar o React

# npm i -g create-react-app

# create-react-app (nome do projeto)
